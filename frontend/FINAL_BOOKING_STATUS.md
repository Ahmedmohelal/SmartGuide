## 🚀 الاستخدام الفوري:

الآن عندما يفتح الدليل أو السائح صفحة الرحلة:

**الدليل يرى:**
```
تفاصيل الرحلة
+ المواعيد المتاحة [جديد]
├─ عرض المواعيد ← رؤية كل المواعيد
└─ إضافة موعد ← إنشاء موعد جديد
```

**السائح يرى:**
```
تفاصيل الرحلة
+ المواعيد المتاحة [جديد]
└─ قائمة المواعيد ← اختر وأحجز
```

## 📊 التدفق الكامل:

```
1️⃣ الدليل ينشئ رحلة جديدة

2️⃣ الدليل يفتح تفاصيل الرحلة

3️⃣ يرى قسم "المواعيد المتاحة"

4️⃣ يضغط على "إضافة موعد جديد"

5️⃣ يملأ: التاريخ + الوقت + الأماكن + السعر

6️⃣ يضغط "إنشاء الموعد"

7️⃣ ✅ الموعد يظهر في القائمة

8️⃣ السائح يفتح نفس صفحة الرحلة

9️⃣ يرى الموعد المتاح

🔟 يختار الموعد ويضغط "احجز"

🔟❶ ✅ الحجز ينجح ويظهر في "رحلاتي"
```
