# نظام إدارة المواعيد المتقدم - دليل الاستخدام

## 🎯 كيف يعمل النظام الآن:

### للدليل (Guide):
عند فتح صفحة تفاصيل الرحلة، سيرى:
1. **تبويب "عرض المواعيد"** - يعرض جميع الـ slots التي أنشأها
2. **تبويب "إضافة موعد جديد"** - نموذج لإنشاء موعد جديد للرحلة
3. أزرار **تعديل** و **حذف** لكل موعد

### للسائح (Tourist):
سيرى فقط **قسم المواعيد المتاحة** مع:
- عرض جميع الـ slots المتاحة للرحلة
- اختيار موعد والضغط "احجز هذا الموعد"

   ├─ الموعد 1 - اختر وأضغط "احجز"

---

## ⚙️ التدفق الكامل:

### 1. **الدليل ينشئ موعد:**
```
الدليل يفتح تفاصيل الرحلة
    ↓
يضغط على تبويب "إضافة موعد جديد"
    ↓
يملأ: التاريخ + الوقت + الأماكن + السعر
    ↓
يضغط "إنشاء الموعد"
    ↓
الموعد يُضاف إلى الـ API
    ↓
الواجهة تتحدث تلقائياً
    ↓
الموعد يظهر في تبويب "عرض المواعيد"
```

### 2. **السائح يحجز موعد:**
```
السائح يفتح تفاصيل الرحلة
    ↓
يرى المواعيد المتاحة
    ↓
يختار موعد بنقرة
    ↓
يضغط "احجز هذا الموعد"
    ↓
يُرسل الحجز للـ API
    ↓
رسالة نجاح
    ↓
الموعد يُضاف إلى "رحلاتي"
```
