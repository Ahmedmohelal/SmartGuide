# 🎯 دليل سريع: كيفية البدء الفوري

## للدليل: إضافة مواعيد للرحلة

### خطوات سهلة:

#### 1️⃣ ادخل صفحة الرحلة
```
من الرئيسية → اختر رحلة → انقر عليها
أو
من لوحة التحكم → اختر إحدى رحلاتك → انقر "عرض التفاصيل"
```

#### 2️⃣ ابحث عن قسم "المواعيد المتاحة"
```
سيكون أسفل:
- معلومات الرحلة
- البرنامج والخدمات
```

#### 3️⃣ انقر على "إضافة موعد جديد"
```
┌─ المواعيد المتاحة 📅
├─ [ عرض المواعيد ] [ إضافة موعد جديد ] ← انقر هنا
└─ ...
```

#### 4️⃣ ملء النموذج
```
┌─────────────────────────────────────────┐
│ تاريخ البداية: [📅 اختر التاريخ]         │
│ وقت البداية:   [⏰ اختر الوقت]           │
│ تاريخ النهاية: [📅 اختر التاريخ]         │
│ وقت النهاية:   [⏰ اختر الوقت]           │
│ عدد الأماكن:   [__ أدخل الرقم]           │
│ السعر:        [__ أدخل السعر] اختياري   │
│                                         │
│ [ إنشاء الموعد ]                        │
└─────────────────────────────────────────┘
```

#### 5️⃣ انقر "إنشاء الموعد"
```
✅ سيظهر: "تم إنشاء الموعد بنجاح"
```

#### 6️⃣ الموعد سيظهر في القائمة
```
سيتم الانتقال تلقائياً إلى "عرض المواعيد"
وستشاهد:
┌─ الموعد الذي أنشأته ─────────────────┐
│ البداية: 15/07/2024, 3:00 م          │
│ الحالة: متاح                         │
│ أماكن: 2                            │
│ السعر: $150                          │
│ [ تعديل ] [ حذف ]                    │
└────────────────────────────────────────┘
```

---

## للسائح: حجز موعد

### خطوات سهلة:

#### 1️⃣ ادخل صفحة الرحلة
```
من الرئيسية → اختر رحلة → انقر عليها
```

#### 2️⃣ ابحث عن "المواعيد المتاحة"
```
أسفل معلومات الرحلة
```

#### 3️⃣ اختر موعداً
```
┌─ الموعد 1 ────────────────────────┐
│ 15/07/2024, 3:00 م              │
│ أماكن: 2 | السعر: $150           │
│ [ احجز هذا الموعد ] ← انقر هنا   │
└────────────────────────────────────┘
```

#### 4️⃣ انقر "احجز هذا الموعد"
```
✅ سيظهر: "تم حجز الموعد بنجاح"
```

#### 5️⃣ شاهد الحجز في ملفك
```
من ملفك الشخصي → "رحلاتي"
سترى حجزك الجديد مع كل التفاصيل
```

---

## ⚠️ ملاحظات مهمة

### للدليل:
- ✅ يجب أن تكون مسجل دخول كـ "دليل"
- ✅ المواعيد تُضاف لرحلتك الخاصة فقط
- ✅ يمكنك حذف أي موعد لم يتم حجزه بالكامل
- ✅ السعر اختياري (إذا تركته فارغاً يصبح مجاني)

### للسائح:
- ✅ يجب أن تكون مسجل دخول كـ "سائح"
- ✅ لا يمكنك الحجز إذا كانت جميع الأماكن محجوزة
- ✅ سيظهر حجزك في "رحلاتي" فوراً
- ✅ حالة الحجز: قيد الانتظار (يتأكدها الدليل)

---

## 🔧 استكشاف الأخطاء

### المشكلة: لا أرى "المواعيد المتاحة"
```
الحل:
1. تأكد من تحديث الصفحة
2. تحقق من تسجيل الدخول
3. جرّب متصفح آخر أو حذف الـ cache
```

### المشكلة: رسالة خطأ عند الإنشاء
```
الحل:
1. تأكد من ملء جميع الحقول المطلوبة
2. تحقق من صحة التاريخ (يجب أن يكون المستقبل)
3. جرّب إعادة المحاولة
```

### المشكلة: لا أستطيع الحجز
```
الحل:
1. تأكد من توفر أماكن
2. تحقق من تسجيل الدخول كـ "سائح"
3. جرّب موعد آخر
```

## 📞 معلومات إضافية

- **Format التاريخ:** YYYY-MM-DD (في منتقي التاريخ)
- **Format الوقت:** HH:mm (24 ساعة)
- **صيغة API:** MM-DD-HH:mm (يتم التحويل تلقائياً)
