## 🔄 التدفقات

### تدفق الدليل:
```
1. تسجيل دخول كـ "guide"
2. فتح صفحة الرحلة
3. العثور على "المواعيد المتاحة"
4. الضغط على "إضافة موعد جديد"
5. ملء البيانات
6. الضغط "إنشاء الموعد"
7. ✅ الموعد يظهر في القائمة
8. يمكن الحذف/التعديل
```

### تدفق السائح:
```
1. تسجيل دخول كـ "tourist"
2. فتح صفحة الرحلة
3. العثور على "المواعيد المتاحة"
4. اختيار موعد
5. الضغط "احجز هذا الموعد"
6. ✅ الحجز يتم بنجاح
7. الحجز يظهر في "رحلاتي"
```
